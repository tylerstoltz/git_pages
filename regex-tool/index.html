Common Patterns
Email Address

Date (MM/DD/YYYY)

Quick Reference
Character Classes
\d Any digit (0-9)

\s Whitespace

Quantifiers
* 0 or more
+ 1 or more
? 0 or 1
{n} Exactly n times
{n,m} Between n and m times
