üìö How to Use SQLite Query Builder

1. Define Your Tables

Start by creating tables in the Table Designer tab. Add columns with their data types. You can also load a sample database to experiment with pre-populated data.

2. Build Your Query

Use the Query Builder tab to visually construct your SQL query:

  • SELECT: Choose which columns to retrieve
  • JOIN: Link tables by specifying matching columns
  • WHERE: Add filters to narrow down results
  • GROUP BY / ORDER BY: Organize your results

3. Execute & Test

Go to the Execute & Test tab to run your query against the database. You can edit the SQL directly or load it from the query builder.

üîó JOIN Types

INNER JOIN - Returns only matching rows from both tables
LEFT JOIN - Returns all rows from left table, matching rows from right
RIGHT JOIN - Returns all rows from right table, matching rows from left
FULL OUTER JOIN - Returns all rows when there's a match in either table

üéØ WHERE Operators

= Equal to | != Not equal to | > Greater than | < Less than | LIKE Pattern matching | IN Match any in list
