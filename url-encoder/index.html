üí° Encoding Types Explained
  • Standard (Component) - encodeURIComponent() - Use for encoding query parameters, path segments, or any part of a URL. Encodes everything except: A-Z a-z 0-9 - _ . ! ~ * ' ( )
  • Full URL - encodeURI() - Use for encoding complete URLs. Preserves URL structure characters like : / ? # [ ] @
  • Form (+ for spaces) - application/x-www-form-urlencoded - Traditional HTML form encoding where spaces become + instead of %20
